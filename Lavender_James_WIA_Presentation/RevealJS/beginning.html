Tipsy.js

Tipsy is a jQuery plugin for creating tooltip effects based in an anchor tag's title attribute

Usages

Tipsy is great for helping users fill out forms and find the better portions of information where theres huge amounts of texts. Its extremely great for Interface usability.

How to write it

This is how you would use/write it

Default

Default location is underneath the selected element and centered

$('#example-1').tipsy();

Gravities

Gravities are used to control the positioning relative to what it is pointing to

$('#foo').tipsy({gravity: 'n'}); // nw | n | ne | w | e | sw | s | se

Fading

It allows for a fading effect to be applied to the tooltip for comfort of visual effects

$('#example-fade').tipsy({fade: true});

Implimentation

Now how should we use this?

Form Fields

This I believe is the best use for them

$(function() {$('#focus-example [title]').tipsy({trigger: 'focus', gravity: 'w'});});

Toggled tooltip

This is great to help users ready certain parts of large bodies of text

onclick='$("#manual-example a[rel=tipsy]").tipsy("show"); return false;'>Show $('#manual-example a[rel=tipsy]').tipsy({trigger: 'manual'});

Code Summary

$.fn.tipsy.defaults = {
delayIn: 0, // delay before showing tooltip (ms)
delayOut: 0, // delay before hiding tooltip (ms)
fade: false, // fade tooltips in/out?
fallback: '', // fallback text to use when no tooltip text
gravity: 'n', // gravity
html: false, // is tooltip content HTML?
live: false, // use live event support?
offset: 0, // pixel offset of tooltip from element
opacity: 0.8, // opacity of tooltip
title: 'title', // attribute/callback containing tooltip text
trigger: 'hover' // how tooltip is triggered - hover | focus | manual
};
